# 🤖 Анонимный Telegram Бот (Render Edition)

Бот для получения полностью анонимных сообщений. Оптимизирован для бесплатного хостинга на Render.

## ✨ Возможности

- 🔒 **Полная анонимность** — отправитель полностью скрыт
- 📱 **Все типы контента** — текст, фото, видео, голосовые, стикеры
- 👑 **Админ-панель** — статистика, управление сообщениями
- 💬 **Ответы на сообщения**
- 🗑 **Удаление сообщений**
- 📊 **Статистика**
- 🛡 **Блокировка**
- 🌐 **Web-сервер** — keep-alive для Render (не даёт уснуть)

## 🚀 Установка на Render

### 1. Подготовка
1. Создай бота в [@BotFather](https://t.me/BotFather) и получи токен
2. Узнай свой ID в [@userinfobot](https://t.me/userinfobot)

### 2. GitHub
1. Создай новый репозиторий на GitHub
2. Загрузи все файлы из этого архива

### 3. Render
1. Зайди на [render.com](https://render.com) и зарегистрируйся
2. Нажми **"New +"** → **"Web Service"**
3. Подключи свой GitHub репозиторий
4. Настройки:
   - **Name:** `anon-bot` (или любое)
   - **Runtime:** `Python 3`
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `python bot.py`
   - **Plan:** `Free`
5. Перейди в **Environment** и добавь переменные:
   - `BOT_TOKEN` = твой токен от @BotFather
   - `ADMIN_ID` = твой ID (только цифры, без кавычек)
6. Нажми **Create Web Service**

### 4. Готово!
Бот запустится через 2-3 минуты. Проверь логи в Render, если что-то не так.

## ⚠️ Особенности Render Free

- Бот "засыпает" после 15 мин бездействия
- Первое сообщение после сна может задержаться на 30-60 секунд
- Web-сервер в `bot.py` помогает Render понять, что сервис живой
- База данных (SQLite) сохраняется между перезапусками
